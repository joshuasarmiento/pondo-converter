# Pondo Converter

Translates Philippine government contract amounts into everyday equivalents — sacks of rice, classrooms, textbooks, and more.

## Folders

- [`frontend/`](./frontend) — Vue 3 + Vite web app
- [`backend/`](./backend) — Express API + Prisma + Turso DB

## Quick Start

```bash
# Backend
cd backend && bun install && bun dev

# Frontend
cd frontend && bun install && bun dev
```

## Live

- Frontend: [pondo-converter.vercel.app](https://pondo-converter.vercel.app)
